# Arka Plan Görseli

Bu klasöre basketbol temalı arka plan görselini ekleyin.

**Dosya adı:** `basketball-bg.jpg`

Görsel, basketbol topunun potadan geçiş anını gösteren bir görsel olmalıdır.

**Nasıl Eklenir:**

1. Basketbol temalı görselinizi bulun veya indirin
2. Dosya adını `basketball-bg.jpg` olarak kaydedin
3. Bu görseli `web/static/img/` klasörüne koyun

**Önerilen özellikler:**
- Çözünürlük: En az 1920x1080 (Full HD)
- Format: JPG (önerilen) veya PNG
- Boyut: Optimize edilmiş (1-3MB arası)
- Konu: Basketbol topunun potadan geçiş anı

**Görsel bulabileceğiniz kaynaklar:**
- Unsplash: https://unsplash.com/s/photos/basketball-hoop
- Pexels: https://www.pexels.com/search/basketball/
- Pixabay: https://pixabay.com/images/search/basketball/

**Not:** Görsel eklenmezse, sayfa otomatik olarak gradient arka plan kullanacaktır.
